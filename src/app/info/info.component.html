How to use the website

On the modules finder page you will find every module that is currently running. You can use the filter to find specific modules and then add them to the your course planner. The modules that you add to your course planner will stay there until you remove them. You can then use the course planner to be informed any modules that clash (eg if you've taken too many credits in one semester). You can find more information about modules including reviews, by clicking on any of the modules on the module finder page.

FAQ's

How to find information about a specific module

Use the search bar at the top of the module finder, You may need to scroll up! From the module finder you can see the pre-requisites and the level of coursework You can then click on the module to see further information such as reviews and lecturer details

How to find next semesters modules

First you should filter by module level, 100 = year 1, 200 = year 2 etc. Then filter by either semester 1 or 2. Finally select the relevant department(s)

How to add a course to course planner

You can either add modules to selected courses on your course planner or by clicking on the '+' button on a module on the module finder page. Once you have modules selected the planner will warn you if you haven't selected the pre-requisite modules or if you do not have enough credits.
